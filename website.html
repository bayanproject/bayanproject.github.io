Projects

  • NBA Predictor - A machine learning project to predict the outcome of NBA games. I use random forests and currently have a test accuracy of 72%. I might make this a mobile application in the future.

  • Grapevine - An UI/UX research project to create an online learning platform for grade school students. Made a high fidelity prototype using Figma. This project was undertaken with 5 other teammates Riya, Rogan, Winnie, Bill and Jack under the guidance of Prof Illona Posner.

  • Intellitable - A javascript library to efficiently represent data tables and create barplots and lineplots using those tables.

  • Food game - A three level mobile game application developed in Java using Android Studio.

  • Sokoban heuristic - An AI project implementing Anytime greedy best-first search and Anytime weighted A* search to solve a sokoban puzzle.

  • Othello AI - An AI project implementing Minimax and Alpha Beta prunning to find the best strategy for the game of Othello.

  • Tenner grid CSP - An AI project using constraint satisfaction to solve the tenner grid problem. This can be extended to solve Sudoku.

  • Jump Game - Doodle Jump game created in Assembly language. Only because this was a course requirement. I will not be going anywhere near Assembly to create games in the future.

  • Predicting diagnostic test answers - A machine learning project to predict whether a student can correctly answer a specific diagnostic question based on the student's previous answers to other questions and other student's responses. Data used is from Eedi, an online education platform. The algorithms we used were k-Nearest Neighbor, Item Response Theory, Matrix Factorization and Random Forests. Test accuracy of 70%.

  • POS tagger - An AI project implementing a Hidden Markov Model to accomplish Part of Speech tagging.
