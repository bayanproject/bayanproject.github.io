WHAT IS THE BAYAN PROJECT?

Aim

  • To develop a new reliable, open-source, and reproducible algorithm for a robust and theoretically grounded detection of communities, with optimal guarantees.

Community Detection

  • The process of inductively identifying groups within a network (graph), is a core problem in data science (link)
  • There are several existing methods and algorithms for approaching this computational problem. One family of algorithms attempts to find communities by maximizing modularity.

Modularity Maximization

  • Modularity: A network-level quantity indicating the fitness of inferred communities in being internally cohesive and mutually separated. (link)
  • Despite their name and design philosophy, current modularity maximization algorithms fail to maximize modularity or guarantee proximity to a globally optimal solution. They result in degenerate solutions of communities that are likely to be far from the intended modular structure with maximum likelihood based on a degree-preserving null model

CURRENT WORK

  • In the first step of this project, we used an exact integer programming model alongside existing heuristic methods for maximizing modularity in small real social networks and random networks
  • We evaluated the gap in modularity between the integer programming solution and four heuristics, CNM, Louvain, Combo and Leiden
    The optimality gap for Belief and NMF were larger than X% and hence have been ommitted from this plot.
    The best performing algorithm, Combo has an optimality gap of X%

  • We also measured the level of similarity between sub-optimal and optimal communities using the concept of normalized mutual information (NMI) which takes the value 1 if the communities match.
  • It is clear that there is a gap between the currently available best methods and the optimal solution. Additionally, the memory and time requirements for the IP make it an impractical solution for large networks. Hence, we are working on the Bayan Algorithm, combining ideas from integer programming (the branch-and-cut algorithm) with efficient heuristic methods to exactly solve this fundamental problem for small and mid-size networks and provide solutions with approximation guarantees for large networks.
